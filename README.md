Путь витязя
===========

Описание
--------

Игра «Путь витязя» предполагалась как игра жанра MMORPG без элементов доната, которые компенсируются за счёт ненавязчивой рекламы, отключаемую по подписке – единственная платная функция этой игры. Данная игра была представлена на Евразийском экономическом форуме молодёжи (ЕЭФМ) в 2024 году на Конгрессе развития игровой индустрии.

Многие онлайн-игры предполагают «донат» – некоторые плюшки в игре за реальные деньги (игровая валюта, всяческие наборы из тех вещей, которые сложно или невозможно набрать, играя без доната, и т.д.), что создаёт неравные условия – те, кто побогаче и могут себе позволить много тратиться, зачастую получают более высокие преимущества по сравнению с теми, кто играет честным трудом, без доната.

Разрабатывается на стеке технологий Java и MySQL и состоит из трёх частей – базы данных, серверное приложение (daemon, он же выполняет функции бэкенда) и клиентское приложение (фронтенд), представляющее собой интерфейс по взаимодействию с сервером.

Есть отдельные базы данных для каждого из т.н. серверов («княжеств»), где хранится информация, актуальная в рамках сервера, и основная база данных с данными, не зависящими от номера сервера. Все базы данных прилагаются в репозитории.

Планировалось разработать интерфейс для взаимодействия с серверным приложением и, через него, базами данных. Но в связи с возникшими трудностями в разработке игры до открытия ЕЭФМ 2024 г., где она была представлена, и с последовавшем за ЕЭФМ нехваткой времени и средств для дальшейшей разработки, к сегодняшнему дню удалось реализовать только часть из того интерфейса, задуманного ранее в концепции игры. В будущем планируется переписать фронтенд на более универсальный в плане платформ движок Unity и доработать демон, добавив туда механику PvP-боёв, до сих пор не реализованную.

По моему мнению, концепция этой игры позволит тем игрокам, которые не имеют возможности донатить в более известных онлайн-играх, не быть униженными теми, кто эту возможность имеют.

А за счёт ненавязчивости рекламы предполагалось сделать аудиторию этой игры куда более широкой - навязчивая реклама, открывающаяся, как правило, во весь экран, раздражает игроков, особенно если она с громким звуком, что особенно доставляет тех, кто играет ночью.
